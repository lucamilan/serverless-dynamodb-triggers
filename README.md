# serverless-spikes
DynamoDb Stream/Events with Serverless.

1 - Clone the repo.
2 - npm install
3 - sls deploy -v

Test it!
Log into the AWS Console, go to your DynamoDB table and insert an item in the table you created. 
After go to CloudWatch you should see the event output in your Logs.
